# 舒尔特方格 H5

[English](https://github.com/Alice-Drop/schulte_grid/README.md) | 简体中文 

本项目是一个基于HTML5的舒尔特方格测试，用于训练和检测用户的注意力与反应速度。可用于认知训练。



在线使用：https://schulte.alicedrop.site/

你也可以把这个项目上传到你的网页服务器，或者直接在本地运行index.html。
